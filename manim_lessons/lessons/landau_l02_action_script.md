# Landau Mechanics 02 — The Principle of Least Action

## 繁體中文旁白

上一集，我們用廣義座標描述了單擺。現在把問題倒過來：如果只知道起點和終點，中間的路徑會是什麼？

從同一個起點到同一個終點，其實有無數條路徑。直線、彎曲的路，甚至來回繞一圈，都可以畫出來。

但真實的粒子不會任意選一條。自然界會挑出一條讓某個總分數保持穩定的路徑。這就是作用量。

在經典力學裡，每一個瞬間的分數叫做拉格朗日量：動能減去位能，也就是 L 等於 T 減 U。

把整段時間裡的 L 全部累積起來，就得到作用量 S，寫成 L 對時間的積分。

我們稍微扭動真正的路徑，讓起點和終點不變。真正的路徑不是讓 S 一定最小，而是讓一階變化消失：δS 等於零。

對所有可能的小扭動都要求 δS 等於零，積分中的係數就必須消失。這一步正好產生 Euler–Lagrange 方程。

所以運動方程不只是描述一個已知的軌跡；它可以從整個路徑的作用量中，被推導出來。這就是最小作用量原理的力量。

## English narration

Last time, we used a generalized coordinate to describe a pendulum. Now turn the question around: if we know only the beginning and the end, what happens in between?

There are infinitely many paths between the same two points. A straight line, a curved line, or even a path that loops around can all be drawn.

But a real particle does not choose arbitrarily. Nature selects a path that makes a certain accumulated score stationary. That score is the action.

In classical mechanics, the score at each instant is the Lagrangian: kinetic energy minus potential energy, L equals T minus U.

Add L over the entire time interval and we obtain the action S, written as the integral of L with respect to time.

Now bend the path slightly while keeping both endpoints fixed. The true path does not necessarily make S smallest; it makes the first change vanish: delta S equals zero.

Demanding delta S equals zero for every small bend makes the coefficient inside the integral vanish. That step produces the Euler–Lagrange equation.

So an equation of motion is not merely a rule for an already known trajectory. It can be derived from the action of every possible path. That is the power of the principle of least action.
